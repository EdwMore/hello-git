# 🚀 Proyecto de práctica con Git y GitHub

Este repositorio fue creado como parte de mi aprendizaje en Git y GitHub, donde practiqué los fundamentos del control de versiones y trabajo colaborativo.

## 📌 Contenido
- Inicialización de repositorio
- Uso de commits
- Manejo de ramas (branches)
- Merge de ramas
- Resolución de conflictos

## 🛠️ Tecnologías utilizadas
- Git
- GitHub

## 🎯 Objetivo
El objetivo de este proyecto fue entender y aplicar buenas prácticas en el uso de Git para el desarrollo de software.

## 📚 Lo que aprendí
- Cómo versionar proyectos correctamente
- Trabajar con ramas para nuevas funcionalidades
- Resolver conflictos al hacer merge
- Subir proyectos a GitHub
